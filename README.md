# Sembrando Esperanza Web

Página pública de la comunidad cristiana **Sembrando Esperanza 🕊**.

## Secciones incluidas

- Inicio tipo revista cristiana.
- Planes bíblicos con lectura por días.
- Historias y reflexiones con lector tipo libro.
- Versículos por tema.
- Álbumes de imágenes cristianas.
- Videos mediante enlaces externos.
- Oraciones.
- Card para llevar a Biblia Universal en Appsem Store.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Variables de entorno

Copia `.env.example` como `.env` y agrega los datos de Firebase cuando tengas listo el proyecto.
